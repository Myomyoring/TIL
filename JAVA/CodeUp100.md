이스케이프(escape) 문자<br><br>


    print( ) 함수를 이용해 문장을 출력할 때, 줄을 바꾸기 위해서는
    '\n'(new line을 의미)을 줄을 바꿀 위치에 넣어야 한다.<br>
    
    '\t': 탭(tab)간격으로 띄어쓰기<br>
    '\\': \ 백슬래시처럼 생겼지만 원화기호<br>
    '\'': ' 작은따옴표<br>
    '\"': " 큰따옴표
